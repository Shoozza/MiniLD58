Balldr
======

Balldr is Pong + Arkanoid + you control the ball.


Controls:
---------
W or Up    - Move up
S or Down  - Move down
A or Left  - Choose left direction after spawn
D or Right - Choose right direction after spawn
P          - Pause game
ESC        - Quit game


Compiling
---------
Install fpc (Freepascal) or Delphi
On Linux: Install liballegro5-dev
On Windows: get allegro-5.0.11-monolith-mt.dll (allegro.cc has precompiled dlls rename 5.0.10->5.0.11)

To compile:
make
or
make delphi
